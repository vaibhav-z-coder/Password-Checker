# 🔐 Password Validation & Visibility Toggle (JavaScript)

## 📌 Project Overview

This project demonstrates a simple **Password Validation System** built using **HTML, CSS, and JavaScript**.

It includes:

* ✅ Password & Confirm Password matching
* 👁️ Show/Hide password (eye toggle)
* ⚡ Real-time validation feedback

---

## 🚀 Features

* 🔹 **Password Matching**

  * Checks if both passwords are identical
  * Displays success or error message dynamically

* 🔹 **Visibility Toggle**

  * Toggle password visibility using eye icon
  * Works for both password fields

* 🔹 **Real-Time Validation**

  * Uses `input` event for instant feedback

* 🔹 **Clean UI Feedback**

  * Green message → Match ✅
  * Red message → Not Match ❌

## ⚙️ How It Works

1. User enters password in first field
2. User confirms password in second field
3. JavaScript compares both values
4. Displays:

   * ✔ Match → Success message
   * ❌ Not Match → Error message
5. Eye icon toggles password visibility
